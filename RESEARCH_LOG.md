# Research Log: Lifestyle Predictors of Type 2 Diabetes

**Тема:** Какие поведенческие и демографические факторы образа жизни наиболее сильно предсказывают наличие диабета 2 типа?

**Датасет:** BRFSS 2015 (Kaggle preprocessed, ~253k строк, 21 признак)  
**Стек:** Python 3.9 · pandas · scikit-learn · XGBoost · LightGBM · SHAP · seaborn  
**Цель:** Статья в медрXiv / PLOS ONE / Frontiers in Public Health

## Итерация 2 - Предобработка и Baseline (Логистическая регрессия)

**Дата:** 2026-05-08  
**Скрипт:** `notebooks/run_iter2.py`

### Выполненные шаги:
- [x] Train/Val/Test Split (70/15/15)
- [x] Нормализация `BMI`, `MentHlth`, `PhysHlth` (StandardScaler)
- [x] Балансировка классов на Train с помощью SMOTE
- [x] Обучение логистической регрессии (Baseline)
- [x] Визуализация ROC-AUC, PR-AUC, Confusion Matrix

### Размеры выборок
- Train: 177,576 строк (После SMOTE: **305,668** строк - классы сбалансированы 1:1)
- Validation: 38,052 строк
- Test: 38,052 строк

### Результаты Baseline-модели (Logistic Regression) на Validation

| Класс | Precision | Recall | F1-Score |
|-------|-----------|--------|----------|
| 0 (Нет диабета) | 0.95 | 0.73 | 0.82 |
| 1 (Диабет) | 0.31 | **0.76** | 0.44 |

- **ROC-AUC:** 0.8192
- **PR-AUC:** 0.3987

*Вывод по Baseline:* SMOTE отлично сработал - мы "научили" модель находить больных диабетом (Recall = 76%), но пока за счёт большого числа ложных срабатываний (Precision для 1-го класса всего 31%). Логистическая регрессия даёт хорошую "базу" для сравнения (ROC-AUC почти 0.82).
